# Hello World Landing Page

## Project Overview
A simple, fully self-contained static landing page.

## Deployment
- **Tier**: 1 (Static — S3)
- **Build output**: `dist/index.html`

## Architecture
- Single `index.html` file with all CSS inlined — no external dependencies.
- Dark theme with a gradient heading.
- Responsive via `clamp()` font sizes.

## Files
- `index.html` — source page
- `dist/index.html` — deployment artifact
